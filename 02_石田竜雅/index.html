2DFighters

HTMLで動作するオリジナル格闘ゲーム。CPUとの1対1のバトルを楽しめます。

🎮 概要

2DFighters は、HTML5 Canvas で動作する2D格闘ゲームです。
森の中を舞台に、プレイヤーとCPUが1対1で戦います。
シンプル操作で遊べるように設計されており、初心者でもすぐに楽しめます。
相手のHPを削り切るか、制限時間内でより多くのダメージを与えた方が勝利となります。

🕹️ 操作方法

  • A / D：移動
  • Space：ジャンプ
  • P：パンチ
  • K：キック
  • O：ガード（ゲージMAX時）
  • 右手の人差し指を[K]、中指を[O]、薬指を[P]に置いて操作するのをおすすめします。

🔥 アピールポイント

  • 森のステージを完全に Canvas で描画したオリジナル背景
  • パンチ・キック・ガード・カウンターが揃った本格的な格闘システム
  • 攻撃ごとのクールダウンやヒットストップで“格闘ゲームらしい重み”を再現
  • CPUは距離を見て接近し、ランダムでジャンプ・ガードを使う簡易AI搭載
  • 30秒の制限時間で勝敗が決まるテンポの良いラウンド制

🛠️ 工夫した点

  • 攻撃判定は矩形同士の当たり判定で正確に処理
  • ガード成功時はカウンターが発動し、敵を大きく吹き飛ばす演出
  • 攻撃時の“ヒットストップ”で格闘ゲームらしい爽快感を演出
  • 無敵時間・ガードゲージ・ガード成功判定など細かいシステムを実装
  • キャラクターの動きは facing（向き）で左右反転し自然なアニメーションに

💻 使用技術

🎨 Canvas描画

森の背景・キャラクター・HPバーなどを Canvas API で描画。

⚙️ ゲームループ

requestAnimationFrame による滑らかなゲーム進行。

🥊 当たり判定

矩形判定でパンチ・キック・ガードを正確に処理。

⌨️ 入力処理

キーボードイベントで操作を制御。

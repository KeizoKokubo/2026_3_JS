1対1で戦うターン制オンライン迷路対戦ゲーム。ある時は迷路の設計者となって敵の行く手を阻み、またある時は脱出者となって特殊能力を活かして迷路を突破せよ！

GAME FEATURES

🛡️

創り手と逃げ手の攻防

プレイヤーは「迷路制作者」と「脱出者」の役割をターン毎に交代。自分が作った迷路で相手を惑わせ、相手の作った迷路を攻略を楽しめます。

⚡

逆転を生む特殊能力

「壁を壊す」「壁をすり抜ける」「罠を設置する」など、キャラクターごとに異なる能力。いつ、どこで使うかが勝敗の鍵を握ります。

🌐

オンライン対戦対応

一対一のオンライン通信対戦に対応。1台のPCで2人が交互に操作して競い合えるモードも搭載しています。

CHARACTERS

クラッシャー CRUSHER

破壊と制限を得意とする、真っ赤な姿の重装甲キャラクター。

作成時 一度通ると二度と通れなくなる床を2個設置できる。
脱出時 目の前を塞ぐ壁を1回だけ叩き壊して強引に進むことができる。
ゴースト GHOST

錯覚と透ける体を操る、妖しく紫に揺らめくトリックスター。

作成時 見た目は壁だが実際には通り抜けられる「隠し壁」を2個設置できる。
脱出時 1マス分の薄い壁なら1回だけすり抜けてショートカットできる。
ディフェンダー DEFENDER

視界遮断と絶対防御を誇る、冷徹で青い守護のスペシャリスト。

作成時 周囲5×5マス以外が全て暗闇に包まれる特殊な床を1個設置できる。
脱出時 迷路内の危険なダメージ（罠やビーム）を1回だけ完全に無効化する。

GAME RULES

1

ターン交代制（最大6往復）

攻守交代を1往復とし、最大6往復の計12ターンを競います。お互いに迷路の「作成」と「脱出」を同じ数だけ体験します。

2

ライフ制の勝負（初期ライフ各3）

脱出側がゴールできればライフ維持。ゴールに失敗（タイムアップや死亡）するとライフを1失います。先に相手のライフを0にするか、6往復終了時点でライフが多い方の勝利です。

3

迷路作成時の注意

迷路作成フェーズでは、スタートからゴールまでが物理的に繋がっている必要があります。絶対にクリアできない迷路は作れません。

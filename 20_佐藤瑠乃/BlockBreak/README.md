# ブロック崩しゲーム

## 概要
このプロジェクトは、ブラウザ上で遊べるシンプルなブロック崩しゲームです。
マウスでパドルを動かし、ボールを弾いてブロックをすべて消すことを目指します。

## できること
- 3つのステージを用意
- ステージ1をクリアすると次のステージを選択可能
- スコアと残機の表示
- マウス操作によるパドル制御

## 遊び方
1. ブラウザで index.html を開く
2. 画面をクリックしてゲームを開始する
3. マウスを動かしてパドルを操作する
4. ボールを跳ね返してブロックをすべて消す
5. ステージをクリアすると次のステージへ進める

## 操作方法
- マウス移動: パドルを動かす
- クリック: ゲーム開始 / ステージ選択

## ファイル構成
- index.html: ゲーム本体
- README.md: この説明ファイル

## ステージごとの難易度調整記録
- ステージ1: 基本構成のため、ボール速度は標準、パドルはやや広めで扱いやすい設計
- ステージ2: ブロック配置を密にし、ボール速度を少し上げて難易度を上昇
- ステージ3: パドルを狭め、ボール速度をさらに上げて挑戦しやすい構成に調整

## 補足
- 画面下にボールが落ちるとライフが1減ります
- ライフが0になるとゲームオーバーです
- 全ステージをクリアすると終了です
